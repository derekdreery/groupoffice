SINGLE GROUP-OFFICE MODULE INSTALLATION INSTRUCTIONS

To install this single module add the folder that is in the same directory
as this file to the "modules" directory of your Group-Office installation.

Put the license <modulename>-<version>-license.txt in the modules/<modulename> directory.

Now login or reload your Group-Office and go to:

Admin menu -> Modules -> Install

Select the new module and click 'Ok'.

Logout and login again and the modules should be available.
